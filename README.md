# kilej.github.io
Daniel Kilarski (Kiłej), ur. 1995 w Szprotawie, to podziemny raper z Wiechlic reprezentujący nowoczesny oldschool i west coast. Działa od 2008 roku, słynąc z dopracowanego flow. Założył skład MwS (płyta „Początek Końca” 2010) i dołączył do WZSP. Wydał solowe albumy „W zakazanym dziale” (2020) oraz świetnie przyjęty „Obiegł” (2024)
